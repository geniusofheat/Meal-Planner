📋 Grocery List :

Open !

Features :

  1. Smart Add Tool : +

    Build your grocery list by typing an item name into the text box or speaking into the microphone(🎤) and pressing the add(+) button.

  2. Smart Lock : 🔒

    When shopping, check the items you purchase. Press the lock button to lock your selections.

  3. Grocery Menu :

    Browse a directory of grocery items within a smart menu to select multiple items without typing.

  4. Grocery Menu Smart Search :

    Search the grocery menu with the text box by typing a name or by speaking into the microphone(🎤).

  5. Save Grocery List :

    You can pin the grocery list to the meal planner for future reference. This allows you to plan a grocery trip for a specific day.

📖 Cookbook :

Open !

Features :

  1. Recipe Cards :

    You have access to hundreds of chef-inspired recipes sorted and alphabetized into categories. Each recipe card opens in its own popup window so you can keep your place in the menu.

  2. Favorite Recipes :

    Add recipes to your favorites list by pressing the star button.

  3. Meal Planner Access :

    Use recipes with the meal planner. Press the plan button within each recipe card.

  4. Recipe Creator :

    Recipe creator lets you create a recipe card with the same cookbook recipe layout. Accessible only in the cookbook.

  5. Recipe Finder :

    The smart search tool lets you search recipes by name or ingredients. Use the text box or the microphone to search the cookbook.

🔎 Recipe Finder :

Search the entire cookbook by recipe name or ingredients. The recipe finder scans every category and highlights every match so you can find exactly what you're looking for.

Open !

♡ Favorites List :

Your personal collection of saved recipes all in one place. Save any recipe from the cookbook to your favorites list and access them any time.

Open !

📅 Meal Planner :

Plan a different meal every day of the month by choosing recipes from the cookbook or your favorites list. You can also create your own recipe cards and save them to favorites for use in the planner.
